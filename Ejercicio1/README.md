#  Eliminador de Espacios en Texto

Este programa en Python lee texto desde la entrada estándar (`stdin`) y elimina todos los espacios en cada línea ingresada.

---

##  Descripción

El script procesa múltiples líneas de texto, eliminando los espacios en blanco dentro de cada línea.  
Si una línea está vacía, se ignora automáticamente.

---

##  Funcionamiento

1. Lee líneas desde la entrada estándar (`stdin`)
2. Elimina espacios al inicio y final (`strip()`)
3. Ignora líneas vacías
4. Elimina todos los espacios dentro del texto (`replace(" ", "")`)
5. Imprime el resultado
